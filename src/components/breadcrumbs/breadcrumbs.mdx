# 面包屑 Breadcrumbs

## 常规用法

<Playground>
  <Breadcrumbs
    breadcrumbs={[
      { text: i18n('百度'), href: 'https://www.baidu.com' },
      { text: i18n('腾讯'), href: 'https://www.tencent.com' },
      { text: i18n('阿里'), href: 'https://www.alibaba.com' },
    ]}
  />
</Playground>

---

## 自定义分隔符

<Playground>
  <Breadcrumbs
    breadcrumbs={[
      { text: i18n('百度'), href: 'https://www.baidu.com' },
      { text: i18n('腾讯'), href: 'https://www.tencent.com' },
      { text: i18n('阿里'), href: 'https://www.alibaba.com' },
    ]}
    separator="-"
  />
</Playground>

---

## 自定义渲染

自定义渲染需要自己去实现默认样式。

<Playground>
  <Breadcrumbs
    breadcrumbs={[
      {
        text: '百度',
        href: 'https://www.baidu.com',
        renderItem() {
          return 'Hello Baidu'
        },
      },
      {
        text: '腾讯',
        href: 'https://www.tencent.com',
        renderItem() {
          return 'Hello Tencent'
        },
      },
      {
        text: '阿里',
        href: 'https://www.alibaba.com',
        renderItem() {
          return 'Hello Ali'
        },
      },
    ]}
  />
</Playground>
